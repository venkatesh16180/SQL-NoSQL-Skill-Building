# MongoDB Assignments (10–14)
**Course: INFO 579 | University of Arizona**

## 📌 Overview
Five progressive MongoDB assignments covering document-oriented 
database design through to advanced aggregation pipelines — 
using a consistent business JSON dataset across all assignments.

## 📋 Assignment Breakdown

| Assignment | Topic | Key Skills |
|---|---|---|
| 10 | Document Data Design | JSON structure, embedded documents, arrays |
| 11 | CUD Operations | insertMany, updateOne, deleteOne, bulk operations |
| 12 | Find Queries | Filters, projections, comparison operators, regex |
| 13 | Aggregation Pipelines | $match, $group, $sort, $project, $unwind |
| 14 | MongoDB Management | Indexes, explain plans, performance optimization |

## 🛠️ Tools
`MongoDB` `mongosh` `JavaScript`

## 📁 File Naming Convention
