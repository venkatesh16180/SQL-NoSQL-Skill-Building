# SQL Assignments (1–9)
**Course: INFO 579 | University of Arizona**

## 📌 Overview
Nine progressive SQL assignments building from data requirements 
analysis through to complex multi-table query operations — all using 
a consistent business database scenario that grows in complexity 
across each assignment.

## 📋 Assignment Breakdown

| Assignment | Topic | Key Skills |
|---|---|---|
| 1 | Data Requirements & ERD | Identifying entities, attributes, relationships |
| 2 | Logical Data Modeling | Translating requirements to logical schema |
| 3 | Conceptual Data Model | Crow's Foot notation, cardinality |
| 4 | Physical Data Model + DDL | Keys, data types, constraints, CREATE TABLE |
| 5 | Basic SQL Queries | SELECT, WHERE, ORDER BY, LIMIT |
| 6 | Intermediate Queries | JOINs, GROUP BY, HAVING, aliases |
| 7 | Advanced Joins & Subqueries | Multi-table JOINs, nested subqueries |
| 8 | Aggregations & Grouping | SUM, COUNT, AVG, GROUP BY, aggregate filters |
| 9 | Complex Multi-table Operations | Combined techniques across full schema |

## 🛠️ Tools
`MySQL` `MySQL Workbench` `SQL` `Crow's Foot Notation`
