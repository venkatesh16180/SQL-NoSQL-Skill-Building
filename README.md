# SQL & NoSQL Skill Building
**Course: SQL/NoSQL Databases (INFO 579) | University of Arizona**

---

## 📌 Overview
A progressive collection of 14 assignments covering the full spectrum 
of relational and non-relational database skills — from data modeling 
and SQL querying to MongoDB document design and aggregation pipelines. 
All assignments are built around a consistent business database scenario, 
allowing skills to compound across each assignment.

## 🗄️ Skills Progression

### SQL (Assignments 1–9)
| Assignment | Topic |
|---|---|
| 1 | Data requirements analysis & basic ERD |
| 2 | Logical data modeling |
| 3 | Conceptual Data Model (Crow's Foot notation) |
| 4 | Physical Data Model + DDL implementation |
| 5 | Basic SQL — SELECT, WHERE, ORDER BY |
| 6 | Intermediate SQL — JOINs, GROUP BY, HAVING |
| 7 | Advanced SQL — subqueries, complex JOINs |
| 8 | Aggregations, window functions |
| 9 | Complex multi-table operations |

### MongoDB (Assignments 10–14)
| Assignment | Topic |
|---|---|
| 10 | Document data design & JSON structure |
| 11 | Create, Update, Delete (CUD) operations |
| 12 | Find queries with filters and projections |
| 13 | Aggregation pipelines |
| 14 | MongoDB database management |
